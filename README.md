# T S Marketing Management System

This project is a React and Firebase-based management system. You can download it as a ZIP file and run it on your local computer.

## How to Run Locally

1. **Required Software:** You must have [Node.js](https://nodejs.org/) installed on your computer.

2. **Install Dependencies:** Go to the project folder in the terminal and type the following command:
   ```bash
   npm install
   ```

3. **Start Project:** Once installed, start the development server with the following command:
   ```bash
   npm run dev
   ```
   Once the server is running, you can see the project at `http://localhost:3000` in your browser.

## Firebase Configuration

This project uses Firebase. The Firebase configuration is provided in the `firebase-applet-config.json` file in the root directory of the project. If you want to use your own Firebase project, update the information in this file from your Firebase console.

## Key Features

- Dynamic loading screen (changeable from admin panel)
- Director and investment management
- Bank transaction tracking
- Super Admin and Admin role-based access control

## Tech Stack

- **Frontend:** React, Tailwind CSS, Lucide React, Motion
- **Backend/Database:** Firebase Firestore, Firebase Auth
- **Build Tool:** Vite
